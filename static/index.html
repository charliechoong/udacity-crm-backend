CRM Backend API

Endpoints

  1. Get all customers: GET /customers
  2. Get customer by ID: `GET /customers/{id}`
  3. Add new customer: `POST /customers`
    Body: { 'name': '{newName}', 'role': '{newRole}', 'email': '{newEmail}' }
  4. Update customer by ID: `PATCH /customers/{id}`
    Body: { 'name': '{newName}', 'role': '{newRole}', 'email': '{newEmail}', 'contacted': 'true/false' }
  5. Delete customer by ID: `DELETE /customers/{id}`
